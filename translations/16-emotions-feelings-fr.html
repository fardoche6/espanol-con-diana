Émotions &
Sentiments

Maîtrisez comment exprimer vos émotions, interroger les autres et répondre avec empathie — avec les structures estar + adjectif et tener + nom.

Édition Française
Phrases Clés — Émotions
😊
estar + adjectif
Estoy feliz
Je suis heureux / heureuse
"Feliz" est invariable en genre — identique au masculin et au féminin. Se prononce : fe-LISS.
😢
estar + adjectif
Estoy triste
Je suis triste
"Triste" ne change jamais — pas de terminaison -a. Même forme pour un homme ou une femme.
😴
tener + nom
Tengo sueño
J'ai sommeil
Comme en français « avoir sommeil » — l'espagnol utilise aussi un verbe avoir (tener) + nom.
😨
tener + nom
Tengo miedo
J'ai peur
Calque parfait avec le français : « avoir peur » = tener miedo. Jamais ~~estoy miedo~~.
😌
estar + adjectif
Estoy tranquilo/a
Je suis calme / serein(e)
Proche de l'espagnol "tranquille". -o pour l'homme, -a pour la femme.
🤩
estar + adjectif
Estoy emocionado/a
Je suis enthousiaste / excité(e)
Accentuation : e-mo-cio-NA-do. Se dit surtout pour une excitation positive — pas "emocionado" pour "émouvant".
🍽️
tener + nom
Tengo hambre
J'ai faim
Calque parfait : « avoir faim » = tener hambre. Erreur classique : ~~Estoy hambre.~~
😠
estar + adjectif
Estoy enfadado/a
Je suis en colère / fâché(e)
Syllabation : en-fa-DA-do. En Amérique latine, on préfère "enojado/a".
😟
estar + adjectif
Estoy preocupado/a
Je suis inquiet / inquiète
Syllabation : preo-cu-PA-do. Même accentuation sur l'avant-dernière syllabe.
🥱
estar + adjectif
Estoy aburrido/a
Je m'ennuie / je suis ennuyé(e)
"Aburrido" = comment on se sent (ennuyé), pas ce qu'une chose est (ennuyeux). Faux-ami potentiel !
🥶
tener + nom
Tengo frío
J'ai froid
Calque parfait : « avoir froid » = tener frío. Deux syllabes : frí-o. Jamais ~~Estoy frío.~~
🌡️
tener + nom
Tengo calor
J'ai chaud
Encore un calque parfait : « avoir chaud » = tener calor. Toujours tener, jamais estar.
Pièges pour les Francophones
⚠️

Le faux-ami de l'accord en genre

En français, "je suis ennuyé(e)" — on accorde parfois, parfois non selon la région. En espagnol, la règle est stricte : adjectifs en -o changent en -a au féminin ; adjectifs en -e ou -z (triste, feliz) restent invariables. Et les noms après tener (hambre, sed, sueño...) ne changent jamais.

Erreur (féminin)
Estoy tristea.
Estoy triste. ✓
Erreur (nom tener)
Estoy hambre.
Tengo hambre. ✓
Accord correct (-o/-a)
Il : cansado
Elle : cansada ✓
La Règle Fondamentale — Estar vs Tener

ESTAR

+ Adjectif · accord en genre
Estoy cansado/a → Je suis fatigué(e)
Estoy triste → Je suis triste
Estoy emocionado/a → Je suis enthousiaste
Estoy preocupado/a → Je suis inquiet(e)
Après estar, le mot décrit comment on est — c'est un adjectif qui s'accorde en genre avec le locuteur.

TENER

+ Nom · jamais d'accord
Tengo sueño → J'ai sommeil
Tengo hambre → J'ai faim
Tengo miedo → J'ai peur
Tengo frío / calor → J'ai froid / chaud
Après tener, le mot est un nom — le français dit aussi "avoir faim / peur / froid." Ces noms ne s'accordent jamais.

Réponses Empathiques
🌿

Bonne nouvelle

Quand quelqu'un dit : "Estoy muy feliz / emocionado..."
  • ¡Qué bien! Super ! Comme c'est bien !
  • ¡Me alegra! Je suis content(e) pour toi !
  • ¡Genial! Génial !
🍂

Mauvaise nouvelle / difficulté

Quand quelqu'un dit : "Estoy muy triste / preocupado..."
  • Lo siento. Je suis désolé(e). / Je compatis.
  • ¡Vaya! Oh là là ! (espagnol européen)
  • ¡Ánimo! Courage ! / Allez !
  • No te preocupes. Ne t'inquiète pas.
Conseils Clés
1
prononciation

Accentuez la bonne syllabe

emo-cio-NA-do · en-fa-DA-do · preo-cu-PA-do — tous accentués sur l'avant-dernière syllabe. "Muy" se prononce « mouï » — une seule syllabe serrée, comme en français « oui » précédé d'un m.

2
piège grammatical

Lo siento ≠ Pardon

"Lo siento" exprime de l'empathie (je le ressens pour toi), pas des excuses. Pour s'excuser, utilisez "perdón" ou "disculpa." En français : "Lo siento" ≈ "Je compatis / je suis désolé(e) pour toi" — pas "désolé d'avoir fait quelque chose."

3
registre

¿Cómo estás? vs ¿Cómo te sientes?

¿Cómo estás? = Comment vas-tu ? (salutation standard). ¿Cómo te sientes? = Comment tu te sens ? (plus intime, quand on voit que quelque chose ne va pas). Même distinction qu'en français entre "ça va ?" et "comment tu te sens ?"
